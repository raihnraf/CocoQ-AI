<task type="auto">
  <name>Task 1: Create charts.js with Chart.js rendering functions for all chart types</name>
  <files>app/static/js/charts.js</files>
  <action>
Create `app/static/js/charts.js` with reusable Chart.js rendering functions.

1. **Chart color constants:**
   ```javascript
   const CHART_COLORS = {
     'Grade A': '#2d6a4f',
     'Grade B': '#e9c46a',
     'Reject': '#e76f51',
   };
   const CHART_DEFAULTS = {
     responsive: true,
     maintainAspectRatio: false,
   };
   ```

2. **`renderGradeDistribution(canvasId, gradeDistribution)` function:**
   - Get canvas by ID, get 2D context
   - Create new Chart with type 'doughnut':
     - labels: ['Grade A', 'Grade B', 'Reject']
     - datasets: [{ data: [gradeDistribution['Grade A'].count, gradeDistribution['Grade B'].count, gradeDistribution['Reject'].count], backgroundColor: [CHART_COLORS['Grade A'], CHART_COLORS['Grade B'], CHART_COLORS['Reject']], borderWidth: 2, borderColor: '#fff' }]
     - options: { ...CHART_DEFAULTS, plugins: { legend: { position: 'bottom' } } }

3. **`renderRejectTrend(canvasId, trendData)` function:**
   - Get canvas by ID, get 2D context
   - Create new Chart with type 'line':
     - labels: trendData.map(d => d.date)
     - datasets: [{ label: 'Reject Rate (%)', data: trendData.map(d => d.reject_rate), borderColor: '#e76f51', backgroundColor: 'rgba(231,111,81,0.1)', fill: true, tension: 0.3, pointRadius: 4, pointBackgroundColor: '#e76f51' }]
     - options: { ...CHART_DEFAULTS, scales: { y: { beginAtZero: true, max: 100, ticks: { callback: v => v + '%' } } } }

4. **`renderFeatureImportance(canvasId, features)` function:**
   - Get canvas by ID, get 2D context
   - Create new Chart with type 'bar', options: { indexAxis: 'y', ...CHART_DEFAULTS } (horizontal bar):
     - labels: features.map(f => f.feature)
     - datasets: [{ label: 'Importance', data: features.map(f => f.importance), backgroundColor: '#2d6a4f', borderRadius: 4 }]
     - options: { ...CHART_DEFAULTS, indexAxis: 'y', scales: { x: { beginAtZero: true, max: 1, ticks: { callback: v => (v * 100).toFixed(0) + '%' } } }, plugins: { legend: { display: false } } }

5. **`async loadAndRenderFeatureImportance(canvasId)` function:**
   - Fetch GET `/api/feature-importance`
   - Call `renderFeatureImportance(canvasId, data.features)`

6. **`async loadAndRenderDashboardCharts()` function:**
   - Fetch GET `/api/stats`
   - If `grade-dist-chart` canvas exists: call `renderGradeDistribution('grade-dist-chart', stats.grade_distribution)`
   - If `reject-trend-chart` canvas exists: call `renderRejectTrend('reject-trend-chart', stats.reject_trend)`

7. **Auto-initialization:**
   - On DOMContentLoaded, call `loadAndRenderDashboardCharts()`
   - On DOMContentLoaded, if `feature-importance-chart` canvas exists, call `loadAndRenderFeatureImportance('feature-importance-chart')`
  </action>

  </verify>
