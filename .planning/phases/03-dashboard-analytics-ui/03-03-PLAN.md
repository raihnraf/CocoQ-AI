  <name>Task 2: Create predict.js with client-side validation, API submission, and result rendering</name>
  <files>app/static/js/predict.js</files>
  <action>
Create `app/static/js/predict.js` with the following functionality:

1. **Validation constants** (mirror server-side validators.py exactly):
   ```javascript
   const RANGES = {
     temperature: { min: 90, max: 140, label: 'Temperature' },
     moisture: { min: 0, max: 10, label: 'Moisture' },
     pH: { min: 2, max: 9, label: 'pH' },
     color_score: { min: 0, max: 100, label: 'Color Score' },
     cooking_time: { min: 30, max: 150, label: 'Cooking Time' },
     dryness_level: { min: 1, max: 5, label: 'Dryness Level' },
   };
   const ALLOWED_SUPPLIERS = ['Central Java', 'Central Kalimantan', 'East Java', 'West Java', 'Yogyakarta'];
   ```

2. **`validateForm()` function:**
   - Check each field in RANGES: value must be a number within [min, max]
   - Check supplier_origin is in ALLOWED_SUPPLIERS
   - Check visual_inspection is one of: 'poor', 'fair', 'good', 'excellent'
   - Return `{ valid: true }` or `{ valid: false, error: 'message' }`

3. **`showError(message)` function:**
   - Set `#form-error` text content, set `display: block`
   - Auto-hide after 5 seconds

4. **`displayResult(data)` function:**
   - Show `#result-card` (set `display: block`)
   - Set `#result-grade` text to data.grade, apply color class:
     - Grade A → `style="color: var(--success)"`
     - Grade B → `style="color: var(--warning)"`
     - Reject → `style="color: var(--danger)"`
   - Set `#result-confidence` to `(data.confidence * 100).toFixed(1)`
   - Set `#result-recommendation` text content to data.recommendation

5. **Form submit handler:**
   - `document.getElementById('prediction-form').addEventListener('submit', async (e) => { ... })`
   - Prevent default
   - Hide result card, hide error
   - Run validateForm(); if invalid, showError() and return
   - Disable submit button, set text to "Predicting..."
   - Build payload object from form fields: `{ temperature, moisture, pH, color_score, cooking_time, dryness_level, supplier_origin, visual_inspection }`
   - Fetch POST `/api/predict` with `{ method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify(payload) }`
   - If response ok: parse JSON, call displayResult(data)
   - If response not ok: parse JSON error, showError(error.message or error.error)
   - Re-enable submit button, restore text to "Get Prediction"

6. **DOMContentLoaded listener:** Attach form submit handler
